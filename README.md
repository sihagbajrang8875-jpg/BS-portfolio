# Bajrang Sihag — Portfolio

This is a minimal React + Vite portfolio scaffold with Tailwind CSS.

Quick start:

```bash
# install
npm install

# run dev
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```

Files added:
- `src/` — React components and styles
- `index.html` — entry
- `tailwind.config.cjs`, `postcss.config.cjs` — Tailwind setup
